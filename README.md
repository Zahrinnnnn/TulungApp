## Quick Start

```bash
git clone https://github.com/Zahrinnnnn/TulungApp.git
cd TulungApp/tulung
npm install
npm start
```

Configure `.env`:
```env
EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
EXPO_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
EXPO_PUBLIC_OPENAI_API_KEY=your_openai_key
```
